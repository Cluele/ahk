AutoHotKey Scripts
==================

### NewFile (Ctrl+Alt+N)

Create a new blank file.

### NewTextFile (Ctrl+Shift+T)

Create a new blank text file.
The difference of "NewFile" is auto appending ".txt" extension.

### bluewind (Win+Space)

Call bluewind with [Win+Space] on Windows 7.
